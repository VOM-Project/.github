# :computer:VOM : WebRTC와 WebPushNotification을 활용한 실시간 웹캠 상호작용 플랫폼

1. 로그인, 회원가입
- 구글소셜로그인
- 닉네임, 생년월일, 사는지역, 관심 키워드 설정
- 로그아웃
    
2. 실시간 화상 채팅 
- 1:1 화상 채팅
- 상대방의 미니홈피에서 화상채팅 요청
- 화상채팅 알림: 웹푸시알림
  
3. 미니홈피
- 앨범 업로드 및 삭제
- 프로필 조회
- 간단한 자기소개글 작성 및 수정
- 인사글 작성 및 수정
- 프로필 수정
- 닉네임 검색
